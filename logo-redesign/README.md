# Logo — V2 (current)

The Run The Numbers® V2 logo system. Canonical decision record lives in Notion: [Logo design — V2 wordmark (decided)](https://www.notion.so/3678c4a62c3b8181b8a9d5187a2562c4). This README is a working summary; if it ever drifts from the Notion page, Notion wins.

## What's here

### SVG (`svg/`)

- `RTN_Logo_Warm_Ink.svg` — Warm Ink `#1B1A17`. Default for Canvas or light surfaces. The everyday version.
- `RTN_Logo_Ledger_Blue.svg` — Ledger Blue `#2B4A7C`. Brand-forward placement: hero sections, primary marketing surfaces.
- `RTN_Logo_Canvas_Cream.svg` — Canvas Cream `#FAF7F2`. Reversed-out for use on Ledger Blue, Warm Ink, or photographic backgrounds.
- `RTN_Logo_Black.svg` — Pure black `#000000`. For mono print, faxes, anywhere colour fidelity isn't guaranteed.
- `RTN_Logo_White.svg` — Pure white `#FFFFFF`. For dark backgrounds where Canvas Cream would look off-brand or wash out.

### PNG (`png/`)

PNG raster exports of the three brand-coloured logos at three sizes:

- `RTN_Logo_{Ledger_Blue,Canvas_Cream,Warm_Ink}_X1.png` — base size
- `RTN_Logo_{Ledger_Blue,Canvas_Cream,Warm_Ink}_X2.png` — 2x
- `RTN_Logo_{Ledger_Blue,Canvas_Cream,Warm_Ink}_X4.png` — 4x

Use SVG by default. PNGs are for placements that can't take vector (email signatures, some doc embeds, social profile pictures).

## The mark in one paragraph

A single stacked lockup. "RTN" set big in Fraunces display cut (opsz 144, weight 500), with "RUN THE NUMBERS®" set small underneath in Geist uppercase, generously tracked. RTN cap height roughly 5x the descriptor cap height. ® lives on the full descriptor string, never on the three-letter mark. No border, no container.
