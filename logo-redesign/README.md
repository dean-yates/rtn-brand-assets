# Logo redesign — working files

Exploration toward the V2 logo. Current production marks live in `../logos/` and are unchanged until this redesign locks.

## What's here

- `test-bench.html` — single-file HTML test bench. Open in any browser to see Stacked Lockup D at proper scale with live Fraunces and Geist from Google Fonts. Renders the master lockup, size cascade, monogram-only variants, favicons at 16/32/64/180px, a mock browser tab, and background context tests across Canvas, Ledger Blue, Warm Ink, and pure white.

## Status

Direction settled on Stacked Lockup D: "RTN" in Fraunces with "RUN THE NUMBERS®" tracked beneath in Geist. Favicon decision: full "RTN" letters on a Ledger Blue rounded square. Next phase is type refinement (kerning, tracking, optical sizing) before production SVG export.
